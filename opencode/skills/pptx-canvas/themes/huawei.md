# Huawei — Technical Report

## Mood

Precise, structured, engineering-oriented. A clean white foundation accented by
Huawei red for emphasis and branding. Content-dense but well-organized with clear
visual hierarchy. Conveys technological rigor and credibility. Suitable for
technical reports, product specs, architecture overviews, and roadmap presentations.

## Colors

- Primary: `#CE0E2D` — Huawei red, used for key accents, titles, and emphasis
- Secondary: `#C7000B` — darker red variant for hover / active states, depth
- Accent: `#E63E30` — warm red-orange, for secondary highlights and icons
- Tech Blue: `#0060AD` — used in technical diagrams, links, secondary data
- Background Light: `#FFFFFF` — white, primary slide background
- Background Dark: `#1A1A1A` — near-black, for contrast slides or footers
- Alt Surface: `#F5F5F5` — light gray, content panels and table backgrounds
- Surface Elevated: `#E8EAED` — slightly darker gray, for nested cards
- Text Primary: `#222222` — near-black for body text
- Text on Dark: `#FFFFFF` — white text on dark backgrounds
- Text Secondary: `#666666` — medium gray for secondary info
- Text on Red: `#FFFFFF` — white text on red accent backgrounds
- Border: `#D0D0D0` — light gray for table borders, dividers

## Typography

- h1: 36px, bold (slide title)
- h2: 26px, bold (section header)
- h3: 20px, semibold (subsection)
- Body (p): 15px, regular
- Caption: 12px, regular, Text Secondary color

## Layout Principles

- **Header bar**: a thin red stripe or red-accented header area at the top of each slide with the slide title.
- Content area below the header, typically with left-aligned text.
- Prefer structured layouts: numbered items, specification tables, architecture diagrams.
- Use the Alt Surface color for table cell backgrounds to create visual separation.
- Icons and small illustrations accompany technical points where possible.
- Footer with page number and optional Huawei branding.
