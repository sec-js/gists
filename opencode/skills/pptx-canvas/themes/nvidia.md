# Nvidia — Product Showcase

## Mood

Bold, high-tech, futuristic. A dark foundation illuminated by signature Nvidia green.
Dramatic visual impact — large product images, bold headlines, minimal body text.
Designed for keynotes and product launches: every slide should feel like an unveiling.
Hyper-angular geometry, tight typography, no soft decorative elements.

## Colors

- Primary: `#76B900` — Nvidia green, the single accent color for CTAs, highlights, and key data
- Primary Dark: `#5A8D00` — deeper green for secondary accents, borders
- Background Dark: `#1A1A1A` — near-black, primary slide background
- Background Deeper: `#111111` — true dark for full-bleed image backdrops
- Surface: `#242424` — dark gray, for content cards and elevated panels
- Surface Light: `#2E2E2E` — slightly lighter gray, nested panels
- Background Light: `#F7F7F7` — off-white, occasional contrast slides
- Text Primary: `#FFFFFF` — white, main text on dark backgrounds
- Text on Light: `#1A1A1A` — near-black for rare light slides
- Text Secondary: `#A0A0A0` — muted gray for captions, secondary info
- Text Accent: `#76B900` — green text for highlighted terms, specs
- Hairline: `#404040` — subtle dividers between sections
- Border Accent: `#76B900` — green border for emphasis panels

## Typography

- h1: 48px, bold (product name, hero statement)
- h2: 32px, bold (feature category, spec group)
- h3: 22px, semibold (sub-feature, detail header)
- Body (p): 16px, regular
- Caption: 13px, regular, Text Secondary color

## Layout Principles

- **Dark-first**: most slides use the dark background. Light slides are rare exceptions.
- Large hero images for product shots — let the product dominate the slide.
- Spec callouts: use green accent text or green-bordered panels for key numbers.
- Minimal text per slide. Headlines and a few supporting points at most.
- No decorative gradients, no drop shadows, no rounded corners. Sharp, angular geometry.
- Green is the only accent — used consistently for every interactive or emphasis element.
- 2px borders, sharp corners. Hyper-structured grid layouts.
