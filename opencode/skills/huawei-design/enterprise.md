# Huawei Enterprise Technical Theme

## Aesthetic Direction

Professional, technical, and trustworthy. This theme communicates expertise and reliability through clean data presentation, architectural clarity, and measured sophistication.

## Reference Pages
- `e.huawei.com` - Enterprise main site
- `huaweicloud.com` - Cloud services
- Developer documentation - `developer.huawei.com`, forums

## Visual Characteristics

### Overall Feel
- Technical competence and precision
- Trustworthy and established
- Data-driven and factual
- Professional but not cold
- For developer docs: Code-focused, information-dense, searchable

### Color Treatment
- **Primary**: Corporate Blue `#003087` for authority
- **Secondary**: Teal `#008577` for technical accents
- **Backgrounds**: White `#FFFFFF` to light gray `#F5F7FA`
- **Text**: Dark slate `#1C2A3A` for primary, gray `#5A6978` for secondary
- Accent colors used sparingly for data visualization

### Typography Feel
- Clear, readable, professional
- Slightly technical character (good x-height)
- Effective hierarchy through size contrast
- Monospace for code/technical elements

### Layout Approach
- Structured grid-based layouts
- Clear information architecture
- Card-based feature presentations
- Data visualized clearly
- Comparison tables when relevant
- For docs: Sidebar navigation, content area, TOC

## Key Design Elements

### Header
- White background with subtle shadow
- Logo left, functional navigation right
- Secondary product category navigation (tab-like)
- "Contact Sales" CTA prominent
- For docs: Simpler, search bar prominent

### Hero Section
- Professional gradient backgrounds (blue tones)
- Clear value proposition headline
- Subheadline with details
- Two CTAs: "Contact Sales" and "Learn More"
- Technical illustration or product shot
- For docs: Often skipped or minimal

### Solution Cards
- White background with subtle border
- Icon + title + description layout
- Hover: border color change to blue
- Clean, scannable information

### Statistics Section
- Bold numbers (large, prominent)
- Blue background for impact
- White text for contrast
- 4-column grid for metrics

### Technical Content
- Clean code blocks with dark background
- Clear specification tables (zebra striping)
- Architecture diagrams with labels
- Tabbed content for organization
- For docs: Syntax highlighting, copy buttons

### Developer Documentation Layout
- Left sidebar: Navigation tree
- Main content: Centered, readable width
- Right sidebar: On-page TOC
- Code blocks: Dark theme, language indicator
- Search: Prominent, always accessible

### Case Studies
- Professional imagery
- Industry + company context
- Quantified results/metrics
- Professional presentation

### Contact/CTA Sections
- Blue gradient backgrounds
- White text
- Clear contact form or CTA button
- Trust signals nearby

### Footer
- Dark slate background (`#1C2A3A`)
- Organized multi-column links
- Comprehensive

## Imagery Style
- Technical illustrations and diagrams
- Professional B2B photography
- Clean data visualizations
- Office/infrastructure imagery
- Avoid overly flashy graphics

## Data Presentation
- Clear tables with headers
- Visual statistics and metrics
- Comparison matrices
- Icon-based feature lists
- For docs: Parameter tables, return values

## Animations
- Subtle and professional
- Fade and slide for content
- Quick, not distracting
- Focus on functionality

## Responsive Behavior
- Tables horizontally scrollable on mobile
- Maintain data readability
- Card-based layouts stack naturally
- Clear touch interactions
- For docs: Collapsible sidebar on mobile

## Variant: Developer Documentation
- Sidebar navigation (collapsible)
- Breadcrumb trails
- Code blocks with syntax highlighting
- Copy-to-clipboard functionality
- Version indicators
- Search functionality prominent
- API reference
- Minimal styling imagery, maximum information
