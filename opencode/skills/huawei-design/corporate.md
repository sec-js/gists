# Huawei Corporate/Official Theme

## Aesthetic Direction

Professional, trustworthy, and sophisticated. This theme conveys corporate authority through clean layouts, measured typography, and a refined color palette.

## Reference Pages
- `huawei.com` - Main corporate
- `huawei.com/en/bond-investor-relations` - Investor relations (financial data, structured tables)

## Visual Characteristics

### Overall Feel
- Clean and corporate without being sterile
- Professional gravitas with human approachability
- Information-dense but never cluttered
- For investor relations: data-focused, table-heavy, precise

### Color Treatment
- **Primary**: White backgrounds with Huawei Red (`#CF0A2C`) accents
- **Secondary**: Corporate Blue (`#003087`) for trust signals
- **Neutral**: Grays from `#F5F5F5` to `#1A1A1A` for hierarchy
- Avoid pure black; use `#1A1A1A` or `#333333` instead

### Typography Feel
- **Headings**: Bold, confident, slightly tighter letter-spacing
- **Body**: Comfortable reading with generous line-height (1.6-1.7)
- Use a tall x-height for modern, approachable feel
- For financial data: Monospace for numbers, clear alignment

### Layout Approach
- Centered content containers (max-width ~1200px)
- Clear visual hierarchy through size and spacing
- Card-based content blocks with subtle shadows
- Consistent padding (8px base unit)
- For investor relations: Dense but scannable data tables

## Key Design Elements

### Header
- White background, subtle bottom border
- Logo left, navigation centered or right
- Height around 70px, fixed on scroll with shadow
- Clean mega-menu on hover
- For IR pages: Simpler header, fewer navigation items

### Hero Section
- White or very light gray background
- Centered or left-aligned text
- Professional imagery or abstract corporate graphics
- Keep text concise, impactful
- For IR: Often no hero image, direct to data

### Content Cards
- White background
- Subtle shadow: `0 2px 12px rgba(0,0,0,0.08)`
- 8px border-radius
- Hover: slight lift with deeper shadow
- Clean spacing inside (24px padding)

### Data Tables (Investor Relations)
- Clean borders, minimal vertical lines
- Zebra striping for readability
- Highlighted rows for key figures
- Downloadable format indicators

### Buttons
- **Primary**: Huawei Red background, white text, slight rounded corners
- **Secondary**: White or transparent with blue border
- Generous padding (14px 32px)
- Subtle hover darkening

### Footer
- Dark background (`#1A1A1A`)
- White text with gray secondary text
- Multi-column link layout
- Simple, no-frills

## Imagery Style
- Professional photography with natural lighting
- Abstract geometric shapes for tech feel
- Team photos: candid, warm, not overly posed
- Avoid stock photo clichés
- For IR: Charts, graphs, infographics preferred

## Spacing Philosophy
- Large vertical spacing between sections (80px desktop)
- Cards have breathing room
- Text columns not too wide (60-70 characters optimal)
- For data sections: Tighter spacing acceptable

## Animations
- Subtle and professional
- Fade-in on load (0.5s)
- Gentle hover transitions (0.3s)
- No bouncy or aggressive effects

## Responsive Behavior
- Single column on mobile
- Touch-friendly tap targets (44px minimum)
- Maintain visual hierarchy across sizes
- Tables scroll horizontally on mobile

## Variant: Investor Relations
- More data-focused design
- Structured tables and financial charts
- Downloadable document links prominent
- Less visual imagery, more information density
- Clear date/period indicators
