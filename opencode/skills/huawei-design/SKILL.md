# Huawei Front-End Design Skill

## Overview

This skill provides aesthetic design guidelines for creating static single-file HTML pages that authentically replicate Huawei's distinctive visual language. Focus on the overall look and feel rather than implementation details.

## Theme Selection

Choose the appropriate theme based on your content:

| Theme | Use For | Theme File |
|-------|---------|------------|
| Corporate/Official | Company pages, news, about pages | `corporate.md` |
| Consumer Product | Smartphones, wearables, laptops, tablets | `consumer-product.md` |
| Enterprise Technical | Enterprise solutions, cloud services | `enterprise.md` |
| Technical/Academic | Technical reports, research papers, white papers | `academic.md` |

## How to Use

1. Load this `SKILL.md` file for common design principles
2. Load the relevant theme file for your content type:
   - Corporate/Official → `corporate.md`
   - Consumer Product → `consumer-product.md`
   - Enterprise Technical → `enterprise.md`
   - Technical/Academic → `academic.md`

## Core Design Identity

### Brand Colors

**Huawei Red** (`#CF0A2C`) - Primary accent, CTAs, highlights
**Corporate Blue** (`#003087`) - Trust, enterprise, secondary actions
**Gold** (`#C9A227`) - Premium, luxury products

### Overall Aesthetic

Huawei design emphasizes:
- **Cleanliness**: Generous white space, uncluttered layouts
- **Sophistication**: Subtle gradients, refined typography
- **Balance**: Precise alignment, harmonious proportions
- **Premium feel**: Quality imagery, elegant interactions
